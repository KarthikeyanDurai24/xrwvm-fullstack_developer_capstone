## Project Overview
This project is a full-stack web application developed as part of the IBM Full Stack Software Developer Capstone. It allows users to view car dealerships, view dealer details and reviews, analyze review sentiment using a microservice, register, log in, and post reviews for dealerships.

## Architecture
- Django backend for authentication, APIs, and business logic  
- React frontend for UI  
- Node.js microservice for dealership and review data  
- Python (NLTK) microservice for sentiment analysis  

## Technologies Used
- Python, Django, SQLite  
- React, React Router  
- Node.js, Express  
- Docker, Kubernetes  
- IBM Code Engine  
- GitHub Actions (CI/CD)

## Features
- User registration and login
- View dealerships and dealer details
- View reviews with sentiment analysis
- Post reviews (authenticated users)
- Microservices-based architecture
